# Karthik S Adiga — Portfolio v2

Dark navy + blue portfolio built with React + Vite + Tailwind CSS.

## Stack
- **React 18** + **Vite 5**
- **Tailwind CSS v3**
- Google Fonts: Syne + DM Sans
- Custom cursor with smooth follower
- Scroll-triggered reveal animations
- Marquee ticker
- Fully responsive (mobile + desktop)

## Setup

```bash
npm install
npm run dev
# Opens at http://localhost:5173
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel (free, 2 mins)

1. Push this folder to GitHub
2. Go to https://vercel.com → New Project → Import repo
3. Framework preset: **Vite** (auto-detected)
4. Click Deploy ✅

Your live URL: `karthik-portfolio.vercel.app`

## Customise

| File | What to change |
|---|---|
| `src/components/Projects.jsx` | Add/edit projects, update GitHub links |
| `src/components/Skills.jsx` | Add new skills as you learn them |
| `src/components/About.jsx` | Edit bio text and stats |
| `src/index.css` | Tweak colors via Tailwind classes |
| `tailwind.config.js` | Adjust navy/blue color palette |
